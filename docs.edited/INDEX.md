### GET Endpoints
- `GET /api/addresses` - Get user addresses
- `GET /api/orders` - Get user orders
- `GET /api/b2b/members` - Get team members
- `GET /api/b2b/approvals` - Get pending approvals
- `GET /api/b2b/cost-centers` - Get cost centers
- `GET /api/cart/items` - Get cart items
- `GET /api/wishlist` - Get wishlist
- `GET /api/reviews` - Get product reviews
- `GET /api/webhooks` - Get configured webhooks

### POST Endpoints
- `POST /api/addresses` - Create address
- `POST /api/orders` - Create order
- `POST /api/b2b/members` - Add team member
- `POST /api/b2b/cost-centers` - Create cost center
- `POST /api/cart/items` - Add to cart
- `POST /api/wishlist` - Add to wishlist
- `POST /api/reviews` - Create review
- `POST /api/bulk/import` - Import data
- `POST /api/bulk/export` - Export data
- `POST /api/webhooks` - Create webhook

### PATCH Endpoints
- `PATCH /api/addresses/[id]` - Update address
- `PATCH /api/orders/[orderNumber]` - Update/cancel order
- `PATCH /api/b2b/members/[id]` - Update team member
- `PATCH /api/b2b/approvals/[id]` - Approve/reject order
- `PATCH /api/b2b/cost-centers/[id]` - Update cost center
- `PATCH /api/cart/items/[id]` - Update cart quantity

### DELETE Endpoints
- `DELETE /api/addresses/[id]` - Delete address
- `DELETE /api/b2b/members/[id]` - Remove team member
- `DELETE /api/b2b/cost-centers/[id]` - Delete cost center
- `DELETE /api/cart/items/[id]` - Remove from cart
- `DELETE /api/wishlist/[productId]` - Remove from wishlist
